# pyArtemis-Marksman
A responsive Aim Trainer game using the Pygame library, designed to improve the user's reaction time and accuracy through target-based interaction

# Reaction Time Grading System
| Speed (ms)   | Grade |
|--------------|-------|
| 0-200        | S     |
| 200-350      | A+    |
| 350-600      | A     |
| 600-700      | B     |
| 700-950      | C     |
| 950-1150     | D     |
| >1150        | F     |
