# pyArtemis-Marksman
A simple Python/PyGame implementation of an Aim Trainer that tests your average reaction speed and grades you accordingly.

#Reaction Time Grading System:
--------------------------
|  Speed (ms)  |  Grade  |
|    0-200     |    S    |
|   200-350    |    A+   |
|   350-600    |    A    |
|   600-700    |    B    |
|   700-950    |    C    |
|   950-1150   |    D    |
|   >1150      |    F    |
--------------------------
